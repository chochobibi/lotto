
# 로또 번호 추천기

## 개요

간단한 다크 테마의 웹 애플리케이션으로, 로또 번호 추천을 생성하고 표시합니다. 1부터 45까지의 6개 고유 번호로 이루어진 5개 세트를 생성합니다.

## 프로젝트 구조

*   `index.html`: 주 HTML 파일.
*   `style.css`: 스타일링을 위한 CSS 파일.
*   `main.js`: 기능을 위한 JavaScript 파일.

## 디자인 및 기능

*   **한국적인 다크 테마:** 시각적으로 매력적인 어두운 배경과 밝은 텍스트.
*   **현대적인 UI:** 깔끔한 레이아웃, 현대적인 글꼴 및 스타일 구성 요소.
*   **인터랙티브 버튼:** "글로우" 효과가 있는 새 번호를 생성하는 버튼.
*   **결과 표시:** 생성된 번호는 은은한 그림자가 있는 스타일 카드에 표시됩니다.
*   **반응성:** 레이아웃이 다양한 화면 크기에 맞게 조정됩니다.
*   **색상 구분:** 실제 로또 용지처럼 번호대별로 색상을 다르게 표시합니다.

## 현재 계획

1.  **`index.html`:** 제목, 버튼, 결과 컨테이너가 포함된 기본 HTML 구조를 한국어로 변경합니다.
2.  **`style.css`:** 한국적인 느낌의 다크 테마를 적용하고, 버튼과 결과 컨테이너의 스타일을 지정하고, 반응형 디자인을 추가합니다. 번호대별로 다른 색상을 적용합니다.
3.  **`main.js`:** 버튼을 클릭할 때 로또 번호를 생성하고 표시하는 JavaScript 로직을 수정하여 번호에 따라 다른 CSS 클래스를 추가하도록 합니다.
